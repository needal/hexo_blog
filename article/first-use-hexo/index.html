hexo初使用

12/09/2017

这是我第一篇使用hexo搭建的博客

Hexo 是一个快速、简洁且高效的博客框架。Hexo 使用 Markdown（或其他渲染引擎）解析文章，在几秒内，即可利用靓丽的主题生成静态网页。目前很多都使用它建立博客，类似于PHP的wordpress，毕竟后端人员前端都不好。

这里介绍下hexo安装和使用的命令

hexo init

  1. init a hexo project

    $ hexo init “my Hexo”

hexo new

  1. Create a new post

    $ hexo new “My New Post”

hexo server

  1. Run server

    $ hexo server

hexo generate

  1. Generate static files

    $ hexo generate

hexo deploy

  1. Deploy to remote sites

    $ hexo deploy
