## hexo命令

### hexo init
> init a hexo project
``` bash
$ hexo init "my Hexo"
```

### hexo new
> Create a new post
``` bash
$ hexo new "My New Post"
```

### hexo server
>  Run server
``` bash
$ hexo server
```

### hexo generate
> Generate static files
``` bash
$ hexo generate
```

### hexo deploy
> Deploy to remote sites
``` bash
$ hexo deploy
```
