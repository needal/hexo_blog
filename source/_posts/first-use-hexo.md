# _这是我第一篇使用hexo搭建的博客_
Hexo 是一个快速、简洁且高效的博客框架。Hexo 使用 Markdown（或其他渲染引擎）解析文章，在几秒内，即可利用靓丽的主题生成静态网页。目前很多都使用它建立博客，类似于PHP的wordpress，毕竟后端人员前端都不好。
# hexo命令
## hexo init
> init a hexo project
``` bash
$ hexo init "my Hexo"
```
## hexo new
> Create a new post
``` bash
$ hexo new "My New Post"
```
## hexo server
>  Run server
``` bash
$ hexo server
```
## hexo generate
> Generate static files
``` bash
$ hexo generate
```
## hexo deploy
> Deploy to remote sites
``` bash
$ hexo deploy
```
